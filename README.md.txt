# AI Object Detection Security System

## Overview
Real-time AI object detection system using ONNX and OpenCV. Designed for security applications and deployable on Raspberry Pi.

## Features
- Real-time object detection
- Lightweight ONNX model
- GUI support
- Raspberry Pi compatible

## Tech Stack
Python, OpenCV, ONNX Runtime
